# 동미투어 (DM Tour)

> 동양미래대학교 주변 음식점과 고척스카이돔 일정을 제공하는 편의정보 웹서비스

기상청 초단기실황 API에서 현재 기온과 강수 상태를 조회하고, 음식점 데이터에 등록된 날씨 조건과 비교하여 현재 날씨에 어울리는 음식점을 추천합니다.

## 프로젝트 정보

* **개발 기간**: 2023.09 ~ 2023.12
* **개발 형태**: 3인 팀 프로젝트
* **참여 인원**: 이혜진, 이보경, 민채영
* **담당 역할**:
  * Servlet Controller를 활용한 회원가입·로그인 및 세션 처리
  * DAO와 JDBC 기반 회원정보 조회·수정·삭제 기능 구현
  * 기상청 초단기실황 API 호출 및 날씨 데이터 가공
  * 날씨 조건과 음식점 DB를 비교한 추천 기능 구현

## 기술 스택

* Java
* Servlet / JSP / JDBC
* JavaScript / HTML / CSS
* MySQL
* Apache Tomcat 9.0
* JSTL
* 공공데이터포털 기상청 초단기실황 API

## 주요 기능

* 회원가입, 로그인 및 로그아웃
* 세션 기반 사용자 인증 및 권한 구분
* 마이페이지 회원정보 조회·수정
* 관리자 회원 목록 조회·수정·삭제
* 고척스카이돔 일정 조회
* 현재 날씨 조건 기반 음식점 추천

## 핵심 구현

기상청 초단기실황 API에서 `T1H`, `PTY`, `REH` 데이터를 조회하고 필요한 값만 분류했습니다.

이후 현재 날씨 값과 음식점 테이블의 `weather` 값을 비교하여 조건이 일치하는 음식점 중 두 곳을 무작위로 출력했습니다.

```text
기상청 API 호출
→ 날씨 데이터 분류
→ 음식점 DB 조건 비교
→ 추천 음식점 출력
```

## 프로젝트를 통해 경험한 내용

2023년에 진행한 Servlet/JSP 기반 수업 프로젝트입니다.
Servlet을 통한 웹 요청 처리, 세션 기반 사용자 인증, DAO와 JDBC를 활용한 데이터베이스 연동 구조를 경험했습니다. 또한 외부 API에서 필요한 데이터를 가공하고, 데이터베이스 조건과 결합하여 사용자에게 맞춤형 결과를 제공하는 기능을 구현했습니다.
